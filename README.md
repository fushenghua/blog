内部开源简单博客系统（不是最新）

使用目前主流的Web MVC框架，开发简单博客系统，提供内部技术学习交流平台/功能还在完善中。。

仅供学习和参考

### 特性

* 全站HTTPS
* 响应式布局
* 代码高亮
* 在线构建系统
* 富文本编辑
* 登录注册，权限验证
 
### 原理说明

使用目前主流的Web MVC框架，开发简单博客系统，提供内部技术学习交流平台/功能还在完善中，利用Jenkins API可视化构建系统。。
